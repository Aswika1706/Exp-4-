# Exp - 4 HOSPITAL MANAGEMENT SYSTEM

# AIM:
To study the problem statement, SRS document and draw all the UML diagrams of a Hospital Management System.
# SRS (Procedure):
1. Study and understand the requirements of the hospital management system
2. Identify the users or actors like patient, doctor, receptionist, admin
3. List the main functions such as patient registration, appointment scheduling, billing, medical records
4. Identify main objects or classes like Patient, Doctor, Appointment, Treatment, Bill
5. Define the relationships and interactions among system components
6. Create the workflow or process flow for operations like appointment booking or patient admission
7. Organize the system structure and represent it using UML diagrams
